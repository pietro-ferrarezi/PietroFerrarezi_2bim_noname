# Projeto ainda sem nome

## Expeculações
A ideia é fazer um guia agrícola desenvolvido pra centralizar informações sobre culturas cultivadas no Brasil. O sistema vai permitir consultar dados como época de plantio, tempo de colheita e produtividade média, além de visualizar as culturas associadas a diferentes estados. Para fins de demonstração, vou utilizar os 10 maiores estados produtores do Brasil.
